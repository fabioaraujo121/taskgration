## Installation
Add this line to your application's Gemfile:

```ruby
gem "taskgration"
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install taskgration
```

## Usage

### 1. Creating a task to migrate
```bash
$ rails generate taskgration:create YOUR_TASK_MIGRATION_NAME
```
